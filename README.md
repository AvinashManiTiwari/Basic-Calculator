# 🔢 Basic Web Calculator

A clean and responsive **Basic Calculator** built using **HTML, CSS, and JavaScript**.
It performs standard arithmetic operations with a modern **Glassmorphism UI design**.

---

## 🚀 Live Demo

👉 https://AvinashManiTiwari.github.io/Basic-Calculator-app/

---

## ✨ Features

* ✅ Addition, Subtraction, Multiplication, Division
* ✅ Clear (C) and Delete (⌫) functionality
* ✅ Decimal number support
* ✅ Responsive design (mobile-friendly)
* ✅ Smooth hover effects & modern UI

---

## 🖥️ Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling
* **JavaScript** – Functionality
